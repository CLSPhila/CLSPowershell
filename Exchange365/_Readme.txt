In order to use this, your computer needs to be able to connect to Exchange 365 through powershell.  

http://technet.microsoft.com/en-us/library/jj984289%28v=exchg.150%29.aspx

Different Windows OS require system installation and setup/configuration.

Connect.ps1 contains the sysntax to connect to exchange 365
Disconnect.ps1 contains the sysntax to logout.

ExternalContacts.ps1 users ExternalContacts.csv to create Bulk external contacts

ExchangeUsers.csv used when creating BULK Users from portal.microsoftonline.com. Click on the BULK ADD when creating users.
